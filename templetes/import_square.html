📋 How to export from Square

  1. Log in to your Square Dashboard at squareup.com
  2. Go to Items & Orders → Items
  3. Click Import / Export → Export Library
  4. Download the CSV file Square gives you
  5. Upload that file below

⚠️ Square's CSV has extra header/instruction rows at the top — that's fine, the importer handles them automatically.
