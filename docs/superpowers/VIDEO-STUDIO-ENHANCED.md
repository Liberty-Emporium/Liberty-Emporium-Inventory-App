## New Features Overview

### 1. Custom Text Overlays ✨
Add personalized text to your video ads directly on the frame:

**Call-to-Action Text**
- Display promotional messages like "Visit Liberty Emporium", "Limited Time Offer", etc.
- Positioned at strategic location based on video format
- Automatically wrapped to fit
- Color matches template selection

**Custom Tagline**
- Replace the default store address with custom text
- Perfect for taglines like "Est. in Liberty, NC", "Best Selection in Town", etc.
- Appears as footer text below the store name

### 2. Video Watermark/Logo Branding 🎖️
Protect your videos and maintain brand consistency:

**Logo Upload**
- Upload PNG or JPG logos/watermarks
- Recommended size: 200×100px
- Maximum file size: 5 MB
- Fully transparent PNG recommended for best quality

**Logo Positioning**
- **Top Right** — Default professional position
- **Top Left** — For left-aligned branding
- **Bottom Right** — Subtle bottom placement
- **Bottom Left** — Alternative bottom position

**Logo Sizing**
- **Small** — Subtle branding
- **Medium** — Balanced visibility
- **Large** — Prominent branding

### 3. Seasonal Templates 🎄
Pre-designed templates with color schemes for different seasons and holidays:

**Available Templates:**
- **Default** — Classic gold & navy (Liberty Emporium brand colors)
- **Holiday** — Red & green festive colors
- **Valentine** — Pink & purple romantic vibes
- **Spring** — Fresh green & light background
- **Summer** — Bright orange & ocean blue
- **Fall** — Warm orange & brown autumn tones
- **Black Friday** — High-contrast yellow & black
- **Back-to-School** — Bold blue & white academic theme

**How It Works:**
- Each template automatically adjusts text colors and backgrounds
- Colors are optimized for legibility and brand alignment
- Works seamlessly with custom overlays and watermark
- Template selection encodes into video filename for tracking

### 4. Auto-Social-Media Formatting 📱
Generate videos in the exact dimensions needed for each platform:

**Video Format Options:**
1. **YouTube (1920×1080)** — 16:9 Horizontal
   - Ideal for: Website embeds, desktop viewing
   - Auto-scales all elements for wide screens

2. **Instagram (1080×1350)** — 4:5 Portrait
   - Ideal for: Instagram Feed, carousel posts
   - Vertical layout with top product, bottom text

3. **TikTok (1080×1920)** — 9:16 Vertical
   - Ideal for: TikTok, Instagram Reels, YouTube Shorts
   - Full-screen mobile format

4. **Facebook (1200×628)** — 16:9 Link Preview
   - Ideal for: Facebook shares, link previews
   - Desktop-friendly horizontal format

5. **Square (1080×1080)** — 1:1 Ratio
   - Ideal for: Social media stories, tiles
   - Equal width and height

**Format Features:**
- All elements (text, logo, price) automatically scale to fit
- Layout adapts: horizontal formats keep image on left, vertical layouts stack image on top
- Font sizes adjust proportionally for readability on all screen sizes
- Product image and overlays properly positioned for each platform

## How to Use

### Creating a Seasonal, Platform-Specific Video

1. **Select Products** — Choose up to 10 products from the grid

2. **Pick Seasonal Template** 🎄
   - Holiday season? Select **Holiday** (red & green)
   - Valentine's Day? Select **Valentine** (pink & purple)
   - Sale event? Select **Black Friday** (high-impact yellow & black)
   - Or leave **Default** for year-round branding

3. **Choose Social Media Format** 📱
   - TikTok/Reels? Select **TikTok** (9:16 vertical)
   - YouTube embed? Select **YouTube** (16:9 horizontal)
   - Instagram feed? Select **Instagram** (4:5 portrait)
   - Facebook share? Select **Facebook** (16:9 link)
   - All platforms? Generate multiple videos with different formats!

4. **Choose Video Style**
   - **Slideshow** — Smooth, holds product image
   - **Ken Burns** — Cinematic zoom effect

5. **Select Music**
   - Browse library tracks (play preview with ▶️ button)
   - Or upload your own MP3 (max 20 MB)

6. **Set Duration** — 10–60 seconds (15-30 sec ideal for social media)

7. **Add Custom Text**
   - Call-to-Action: "🏪 $9.99 SPECIAL OFFER!"
   - Tagline: "Shop Local • Support Liberty"
   - *Optional — leave blank to use template defaults*

8. **Add Watermark** *(Optional)*
   - Upload your Liberty Emporium logo
   - Choose position (Top Right recommended)
   - Select size (Medium for balanced look)

9. **Create Video** — Click 🎬 "Create Video Ad"
   - All elements auto-scale to match chosen format
   - Slideshow: ~20-30 seconds generation time
   - Ken Burns: ~40-60 seconds (more processing)
   - Format processing: <2 seconds additional

10. **Download & Use**
    - Download MP4 via "⬇️ Download MP4" button
    - Ready for chosen social platform
    - Filename includes template and format: `video_ad_SKU_holiday_1080x1920_TIMESTAMP.mp4`

## Best Practices by Platform

### YouTube (1920×1080)
- Use **Default** or **Holiday** templates
- Keep CTA short and readable from distance
- 30-60 seconds ideal for pre-roll ads
- Logo top-right, Medium size

### Instagram Reels (1080×1350)
- Use **Spring** or **Summer** for seasonal vibrancy
- Short CTA (15-30 seconds)
- Vertical layout stacks product image on top
- Logo positioned top-corner, Small size for less clutter

### TikTok (1080×1920)
- Use **Black Friday** for promotions or trending template
- Keep text short and punchy
- 15-30 seconds optimal (platforms auto-loop)
- Consider trending sounds in music selection
- Large, bold text at bottom

### Facebook (1200×628)
- Use **Holiday** or **Default** for professional look
- Include full store info in tagline
- 15-30 seconds for link preview
- Logo Small to Medium for link preview viability

### Square Displays (1080×1080)
- Perfect for kiosk/tablet in-store loops
- Use **Default** for consistent branding
- 30-60 seconds for continuous loop
- Medium to Large logo for visibility from distance

## Technical Details

### Responsive Layout System
```
┌─ HORIZONTAL (YouTube, Facebook, Square) ──────────────┐
│ [PRODUCT IMAGE]  │  TITLE, PRICE, CTA ON RIGHT       │
│   65% width      │  Right overlay 35% width            │
└────────────────────────────────────────────────────────┘

┌─ VERTICAL (TikTok, Instagram) ────────────────┐
│         PRODUCT IMAGE (60% height)             │
├────────────────────────────────────────────────┤
│  TITLE, PRICE, CTA                              │
│  Text overlaid on dark background (40% height)  │
└────────────────────────────────────────────────┘
```

### Dynamic Scaling
- Font sizes scale proportionally: `size = base_size × (W / 1280px)`
- Logo sizes adjust: `logo_size × (W / 1280px)`
- Padding and spacing maintain aspect-ratio relationships
- Text wrapping adjusts based on available width

### Color Handling
- Template colors are stored as hex values
- Converted to RGB tuples at runtime for PIL
- Accent colors used for titles, CTAs, emphasis
- Background colors applied to overlay regions
- Ensures color accuracy across all platforms

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Text appears cut off | Text auto-wraps; check character limit in CTA field |
| Logo distorted in vertical | Use square logos (1:1 aspect ratio recommended) |
| Video dimension doesn't match platform | Double-check format selection; re-generate if needed |
| Seasonal colors look wrong | Ensure template is selected; incorrect template won't adjust colors |
| Font scaling looks small | Increase CTA/Tagline text length; proportions auto-adjust |


- **Holiday** template + TikTok format = trending TikTok shop video
- **Black Friday** template + all formats = coordinated flash sale campaign
- **Spring/Summer** templates + Instagram format = aesthetic carousel
- **Default** template + YouTube format = evergreen product overview
