What should Claude base the style on?

When Claude picks the color theme and layout for each ad, what should it use to decide?

A

The Product Itself

Claude reads the product title, category, and description and picks a style that fits — e.g. "vintage toy → warm earthy tones", "jewelry → elegant navy/gold". Each product gets its own personality.

B

You Still Pick the Style

You keep the existing style chips (Elegant, Vivid, Forest, Ocean, Warm) and Claude only writes the text. Style stays your choice.

C

Both — Claude Suggests, You Can Override

Claude recommends a style per product, but the style chips are still shown with Claude's pick pre-selected. You can change it before generating.
