Ads Page — Music & Video Feature

How should the music and video features be organized? Pick the layout that feels right.

A

Expand the Existing Ads Page

Add a Music tab and a Create Video button directly on the current /ads page. Everything lives in one place — generated image ads, music picker, and video ads all together.

  • One URL, one page
  • Tabs or sections separate the three content types
  • Simplest for the user to discover
B

New "Video Studio" Section on Ad Generator

Add a Video Studio panel on the Ad Generator page (/ads generator). Users select products, pick a music track, then hit Create Video Ad. Video results appear below alongside image ads.

  • Music + video selection happens before generating
  • Keeps the workflow together: select → style → music → generate
  • More powerful but slightly more complex UI
C

Separate "Video Ads" Page

A dedicated /video-ads page just for video creation. Users pick products, choose music, set style, and generate MP4 videos. The existing ads page stays for image ads only.

  • Clean separation between image and video ads
  • More room to build out video-specific features
  • Requires navigating to a different page
