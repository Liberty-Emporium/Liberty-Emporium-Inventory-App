Seasonal Sale — How-To Guide

Run a Store-Wide Discount

The Seasonal Sale feature lets you apply a percentage discount to an entire product category — great for holiday sales, clearance events, or weekly specials. The discounted price shows everywhere in the app.

🔒 Only the admin account can activate or stop a sale.
✅
Starting a Sale

How to Activate a Sale

  1. Select a Category from the dropdown — only products in this category will be discounted.
  2. Enter a Discount % (e.g. 20 for 20% off).
  3. Click "Activate Sale".
  4. A yellow banner will appear on the Dashboard letting everyone know the sale is active.
🛑
Stopping a Sale

How to End a Sale

  1. When a sale is active, this page shows a banner confirming the current discount and category.
  2. Click the red "Stop Sale" button.
  3. The discount is removed immediately and prices return to normal.
Only one sale can be active at a time. Start a new one to replace the current sale.
