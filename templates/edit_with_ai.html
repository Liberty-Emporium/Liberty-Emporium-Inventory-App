Step 1 — API Key

Setting Up Your Claude API Key

The AI features use Claude AI — you'll need your own Anthropic API key:

  1. Click "🔑 API Key" in the blue box to expand it.
  2. Visit console.anthropic.com → Settings → API Keys → Create Key.
  3. Paste your key into the field — click "Get an API key →" for a direct link.
  4. The key saves automatically in your browser — you only need to do this once.
  5. Create a free account at Anthropic, then add a small balance to activate your key.
Pricing is pay-as-you-go — photo analysis costs just a few cents each.
🤖
Step 2 — AI Analysis

Analyzing a Photo with AI

  1. Click "Choose File" and select a clear photo of your item.
  2. Click "✨ Analyze Photo".
  3. Claude AI fills in the title, description, category, condition, and a suggested price automatically.
  4. Review the filled fields and adjust anything before saving.
A well-lit photo on a plain background gives the best results.
💸
Step 3 — Cost Tracking

Understanding the Cost Tracker

After each analysis the result shows exactly what it cost:

🪙 This analysis: $0.0008 1,842 in / 312 out tokens

The stats bar tracks session cost and total spent across all visits. Click "💳 View account balance →" to check your remaining credits on the Anthropic console.

🏷️
Step 4 — Basic Info

Basic Information Fields

  • SKU — unique ID for the item (e.g. ITEM-001). Required. Cannot be changed after saving.
  • Title — the product name shown on all pages and listings.
  • Description — full details about the item, condition, history, etc.
SKUs must be unique. You must enter one manually — they are not auto-generated.
💰

💾
Step 6 — Save

Saving Your Product

  1. Upload one or more photos in the Images section.
  2. Review all fields to make sure everything looks correct.
  3. Click "✨ Create Product" (or "💾 Save Changes" when editing).
  4. You'll return to the Dashboard where the product appears in the grid.
A backup of your inventory is created automatically every time you save.
