Import from Square — How-To Guide

Bring Your Square Products In

If you already use Square POS, you can export your item library and import it here in seconds. The importer automatically maps Square's fields to this app's fields.

🔒 Only the admin account can import from Square.
⬛
Step 1 — Export from Square

Getting Your Square CSV

  1. Log in to your Square Dashboard at squareup.com.
  2. Go to Items & Orders → Items.
  3. Click Import / Export → Export Library.
  4. Square downloads a CSV file to your computer.
Square adds extra header rows at the top of the file — that's fine, the importer handles them automatically.
🔄
Step 2 — Field Mapping

How Square Fields Map Here

Item Name → TitleProduct display name
SKU → SKURequired — must be unique
Price → PriceSale price
Categories → CategoryFirst category used
Archived Y → SoldN = Available
Condition → ConditionDefaults to "Good"
⚙️
Step 3 — Import Options

Choosing How to Handle Duplicates

If a product SKU from Square already exists in this app, choose what to do:

  • Skip — leave the existing product unchanged. Safe option.
  • Overwrite — update title, description, price, category, and status. Images and location info are kept.
Use Skip for a first import. Use Overwrite to sync price changes from Square.
🚀
Step 4 — Upload & Import

Running the Import

  1. Click "Choose File" and select the Square CSV you downloaded.
  2. Select your preferred duplicate handling option (Skip or Overwrite).
  3. Click "📥 Import Now".
  4. A summary will appear showing how many products were added, updated, or skipped.
⚠️ Create a backup before importing if you want to be able to undo the changes.
