User Management — How-To Guide

Managing Who Can Access Your Store

This page lets you approve new signup requests, reject unwanted ones, and remove users who should no longer have access.

🔒 Only the Owner (admin) account can see and use this page.
⏳
Section 1 — Pending Approval

✅
Action — Approve

Approving a New User

  1. Find the user in the Pending Approval section.
  2. Click the green ✅ Approve button next to their name.
  3. They are instantly moved to Approved Users and can now log in.
No confirmation dialog — approval is instant.
❌
Action — Reject

Rejecting a Request

  1. Find the unwanted request in Pending Approval.
  2. Click the red ❌ Reject button.
  3. A confirmation dialog appears — click OK to confirm.
  4. The request is deleted permanently.
⚠️ Rejection is permanent. The person would need to sign up again if you change your mind.
✅
Section 2 — Approved Users

Approved Users

Shows everyone who currently has access. The Owner (admin) always appears at the top and cannot be removed.

Action — Remove

Removing an Approved User

  1. Scroll to the Approved Users section.
  2. Click the orange 🗑️ Remove button next to the user.
  3. Confirm the action in the dialog that appears.
  4. The user is removed and can no longer log in.
The admin/Owner account can never be removed from this list.
