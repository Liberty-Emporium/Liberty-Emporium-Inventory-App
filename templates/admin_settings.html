🤖 AI API Key (Unified)

Set your Claude API key here once — all AI features (photo analysis, video voiceover, script rewriting, listing generation) will use this key automatically.

Step 1 of 4

Create a free Anthropic account

  1. Go to console.anthropic.com ↗
  2. Click "Sign Up" in the top right corner
  3. Create an account with Google or email
💡 Free credits: New accounts come with $5 of free API credits — enough for thousands of AI analyses!
Step 2 of 4

Navigate to API Keys

  1. After logging in, look for the left sidebar
  2. Click on "API Keys"
  3. Or go directly to: console.anthropic.com/settings/keys ↗
🗝️ The API Keys section is under Settings in the Anthropic console.
Step 3 of 4

Generate your API key

  1. Click the "Create Key" button
  2. Give it a name (e.g., "Liberty Emporium")
  3. Important: Copy the key immediately — it starts with sk-ant-...
⚠️ You only see the key ONCE! If you lose it, you'll need to create a new one. So copy it right away.
Step 4 of 4

Paste your key below and save

  1. Come back to this page
  2. Paste your key (starts with sk-ant-...) into the field below
  3. Click "Save API Key"
✅ Once set, your key is saved automatically. All AI features work instantly — no restart needed!
