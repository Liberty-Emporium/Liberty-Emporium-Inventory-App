App Settings — How-To Guide

Configure Your App

Settings is where you connect the AI features and payment system. You only need to do this once.

🤖
Step 1 — Claude API Key

Set Your AI API Key

  1. Go to console.anthropic.com and create a free account.
  2. Generate an API key (starts with sk-ant-).
  3. Paste it into the Anthropic API Key field and click Save.
This key powers AI photo analysis and AI ad copy. New accounts get $5 free credits — enough for thousands of analyses.
💳
Step 2 — Stripe Keys

Enable Stripe Payments

  1. Go to dashboard.stripe.com/apikeys.
  2. Copy your Secret Key (sk_live_...) and Publishable Key (pk_live_...).
  3. Add both as environment variables in your Railway project → Variables tab.
Railway resets files on every deploy — keys must be set as Railway env vars to survive restarts.

🤖 AI API Key (Unified)

Step 1 of 4

Create a free Anthropic account

  1. Go to console.anthropic.com ↗
  2. Click "Sign Up" in the top right corner
  3. Create an account with Google or email
💡 Free credits: New accounts come with $5 of free API credits — enough for thousands of AI analyses!
Step 2 of 4

Navigate to API Keys

  1. After logging in, look for the left sidebar
  2. Click on "API Keys"
  3. Or go directly to: console.anthropic.com/settings/keys ↗
🗝️ The API Keys section is under Settings in the Anthropic console.
Step 3 of 4

Generate your API key

  1. Click the "Create Key" button
  2. Give it a name (e.g., "Liberty Emporium")
  3. Important: Copy the key immediately — it starts with sk-ant-...
⚠️ You only see the key ONCE! If you lose it, you'll need to create a new one. So copy it right away.
Step 4 of 4

Paste your key below and save

  1. Come back to this page
  2. Paste your key (starts with sk-ant-...) into the field below
  3. Click "Save API Key"
✅ Once set, your key is saved automatically. All AI features work instantly — no restart needed!

Stripe keys must be set as environment variables in Railway — they can't be saved here because Railway resets files on every deploy.

  1. Go to railway.app/dashboard ↗
  2. Open your project → click your service → go to Variables
  3. Add these two variables:
STRIPE_SECRET_KEY = sk_live_... (from dashboard.stripe.com/apikeys)
STRIPE_PUBLIC_KEY = pk_live_... (same page)

Railway will automatically redeploy after you save — then Buy Now will open Stripe Checkout.
