Backups — How-To Guide

Protecting Your Inventory Data

The Backups page lets you save snapshots of your full inventory. If something ever goes wrong, you can restore any previous backup in one click. Up to 20 backups are kept at all times.

🔒 Only the admin account can access Backups.
🤖
Auto Backups

Automatic Backups

The app automatically creates a backup every time you save or edit a product. You don't need to do anything — it happens in the background.

Each backup file shows the date, time, and file size so you can identify when it was created.

Once 20 backups exist, the oldest one is automatically deleted to make room for the new one.
📦
Manual Backup

Creating a Manual Backup

  1. Click the "📦 Create Backup Now" button in the top-right corner.
  2. A new backup snapshot is created immediately.
  3. It appears at the top of the backup list.
Create a manual backup before making large changes — like importing from Square or bulk edits.
⬇️
Download a Backup

Downloading a Backup File

  1. Find the backup you want in the list.
  2. Click "⬇️ Download" next to it.
  3. The backup CSV file is saved to your computer.

Downloaded backups are standard CSV files you can open in Excel or Google Sheets.

♻️
Restore a Backup

Restoring from a Backup

  1. Find the backup you want to restore.
  2. Click "♻️ Restore" next to it.
  3. A confirmation dialog appears — click OK to proceed.
  4. Your entire inventory is replaced with the data from that backup.
⚠️ Restoring a backup overwrites your current inventory. Create a backup first if you're unsure.
